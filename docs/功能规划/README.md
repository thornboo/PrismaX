## 功能概览

### P0 - 核心功能 (MVP)

- 基础聊天
- 多模型支持
- Ollama 本地模型
- 会话管理
- 消息管理
- 流式响应
- Markdown 渲染
- 主题切换
- 多语言

### P1 - 重要功能

- 知识库 RAG
- 文件上传
- 助手预设
- 快捷键
- 搜索功能
- 数据导出

### P2 - 增强功能

- Agent 系统
- MCP 协议
- 插件系统
- 网页搜索
- TTS/STT
- 话题分支

### P3 - 锦上添花

- 助手市场
- 插件市场
