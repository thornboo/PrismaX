## 最终决策

### 确定的技术栈

| 类别           | 选型           | 说明                     |
| -------------- | -------------- | ------------------------ |
| **桌面框架**   | Electron       | 成熟生态，跨平台 UI 一致 |
| **前端构建**   | Vite 6         | 极速 HMR，开发体验好     |
| **UI 框架**    | React 19       | 最新版本                 |
| **UI 组件**    | shadcn/ui      | 高度可定制               |
| **样式**       | Tailwind CSS 4 | 原子化 CSS               |
| **状态管理**   | Zustand 5      | 轻量 + 持久化            |
| **本地数据库** | SQLite         | better-sqlite3           |
| **ORM**        | Drizzle        | 类型安全、轻量           |
| **AI SDK**     | @ai-sdk/openai | 流式响应支持             |

### 决策理由

选择 **Electron + Vite + React** 的主要原因：

1. **开发效率高** - Vite HMR 极速，开发体验好
2. **技术栈成熟** - React 生态丰富，问题都有解决方案
3. **跨平台一致** - Chromium 渲染，UI 表现一致
4. **纯 TypeScript** - 无需学习 Rust
5. **桌面端专注** - 不需要服务端功能
