## Feature Overview

### P0 - Core Features (MVP)

- Basic chat
- Multi-model support
- Ollama local models
- Session management
- Message management
- Streaming responses
- Markdown rendering
- Theme switching
- Multi-language

### P1 - Important Features

- Knowledge base RAG
- File upload
- Cloud sync
- Assistant presets
- Keyboard shortcuts
- Search functionality
- Data export

### P2 - Enhanced Features

- Agent system
- MCP protocol
- Plugin system
- Web search
- TTS/STT
- Topic branching

### P3 - Nice to Have

- Assistant marketplace
- Plugin marketplace
- Team collaboration
- Monitoring dashboard
