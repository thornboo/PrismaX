# PrismaX 项目概述

> 新一代 AI 聊天助手应用 - 功能完善、部署灵活、体验优秀

---

## 项目愿景

打造一个功能完善、部署灵活的 AI 聊天助手应用，支持：

- **桌面应用** - 本地运行，离线可用，快速启动
- **Web 服务** - 浏览器访问，多用户支持，团队协作
- **移动应用** - iOS/Android 支持（规划中）
- **Docker 部署** - 一键部署，简单运维

---

## 核心功能

### 对话能力

- 多模型对话（OpenAI、Claude、Gemini、本地模型等）
- 流式响应，实时显示
- Markdown 完整渲染（代码高亮、表格、公式、图表）
- 多轮对话上下文管理
- 对话分支与历史管理

### 知识增强

- 知识库 RAG（向量搜索、文档问答）
- 多格式文档支持（PDF、Markdown、Word 等）
- 智能分块与检索优化

### 扩展能力

- Agent 系统（工具调用、自动化任务）
- MCP 协议支持（Model Context Protocol）
- 插件系统（第三方扩展）
- Ollama 本地模型集成

### 部署方式

- 桌面应用（Windows/macOS/Linux）
- Web 服务（Docker 一键部署）
- 多用户支持与权限管理
- 云同步（跨设备数据同步）

### 用户体验

- 快速启动，流畅交互
- 美观现代的界面设计
- 深色/浅色主题
- 多语言支持
- 丰富的快捷键

## 技术选型

| 类别 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Next.js 15 + React 19 | App Router + Server Components |
| UI 组件 | shadcn/ui + Radix UI | 可定制性强，无样式锁定 |
| 样式方案 | Tailwind CSS | 原子化 CSS |
| 状态管理 | Zustand | 轻量级，TypeScript 友好 |
| 桌面框架 | Electron | 跨平台桌面应用 |
| 后端 | Next.js API Routes + tRPC | 端到端类型安全 |
| 数据库 | PostgreSQL + pgvector | 向量搜索支持 |
| 本地数据库 | SQLite | 桌面版轻量存储 |
| 缓存 | Redis | Agent 状态管理 |

详细技术选型分析请查看 [技术选型](./技术选型/) 目录。
